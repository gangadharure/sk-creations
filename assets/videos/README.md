# Drop your video clips here
- `hero.mp4` — used as the background of the homepage hero section.
- `showcase.mp4` — used as the mid-page parallax video band.

Recommended specs:
- 1920×1080 (or 1280×720) MP4 H.264
- 15–25 second loop, silent (the site mutes audio)
- < 10 MB each for fast load

If these files are missing, the site automatically falls back to high-quality poster images.
